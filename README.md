# som-trial

## Install project

```
cp .env.example .env
```
```
composer install
```
```
php artisan migrate
```
```
cd fronted
```
```
cp .env.example .env
```
```
yarn
```
```
yarn build
```
```
php artisan serve
```

## Run tests

Run Php Unit and Feature tests
```
./vendor/phpunit/phpunit/phpunit
```

Run Dusk tests
```
php artisan dusk:chrome-driver 72
```
```
php artisan dusk
```
